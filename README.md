# mybookr

NextJS + TypeScript + TanStack Query + Swagger + Stripe + Clerk

## How to install

- `npm install`
- `cp .envrc.example .envrc` (if you have direnv, otherwise use any other way fitting to your OS to set the environment variables.)

## How to develop

- `npm run dev`

## How to build

- `npm install`
- `npm run dev`
