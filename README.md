# mybookr

NextJS + TypeScript + TanStack Query + Swagger + Stripe + Clerk

## How to install

- `npm install`
- `cp .envrc.example .env` (or `.envrc` if you have direnv)

## How to develop

- `npm run dev`

## How to build

- `npm install`
- `npm run dev`

## How to update TS types + API hooks

- `npm api:generate`
